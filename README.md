# SURVIVAL MODE - 末日生存倒计时

一个具有末日生存风格的倒计时网页应用，帮助你在紧张的氛围中追踪重要目标的剩余时间。

## 预览

网站采用赛博朋克/末日废土风格的视觉设计，配合实时倒计时、进度条和激励语句，营造紧迫感和使命感。

## 功能特性

- **实时倒计时** - 精确显示剩余的天、小时、分钟和秒
- **进度追踪** - 可视化展示任务完成进度百分比
- **生存天数统计** - 记录并显示已坚持的天数
- **激励语句** - 每30秒自动切换的随机激励语
- **日期信息** - 显示任务开始时间、当前时间和截止时间
- **响应式设计** - 完美适配桌面端和移动端设备

## 视觉效果

- 暗黑科技风格界面
- 噪点纹理和扫描线特效
- 闪烁的警告条和边框动画
- 荧光绿/警示红配色方案
- 四角装饰边框

## 使用方法

1. 打开 `index.html` 文件
2. 修改 JavaScript 中的 `CONFIG` 对象来设置你的日期：

```javascript
const CONFIG = {
  startDate: "2026-03-08",  // 开始日期
  endDate: "2026-04-01",    // 结束日期（修改这里！）
};
```

3. 可选：自定义 `motivations` 数组来添加你自己的激励语句

## 技术栈

- HTML5
- CSS3（动画、渐变、响应式布局）
- 原生 JavaScript（无依赖）

## 部署

这是一个纯静态网页，可以直接部署到任何静态网站托管服务：

- GitHub Pages
- Vercel
- Netlify
- Cloudflare Pages
- 或任何 Web 服务器

## 本地运行

直接在浏览器中打开 `index.html` 文件即可，无需服务器环境。

```bash
# 或使用简单的 HTTP 服务器
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

## 自定义

### 修改颜色主题

编辑 CSS 中的 `:root` 变量：

```css
:root {
  --bg-color: #0a0a0a;      /* 背景色 */
  --warning-red: #ff3333;    /* 警示红 */
  --toxic-green: #00ff41;    /* 荧光绿 */
  --amber: #ffaa00;          /* 琥珀色 */
  --gray: #333;              /* 灰色 */
  --light-gray: #888;        /* 浅灰 */
}
```

### 添加激励语句

在 `motivations` 数组中添加新的语句：

```javascript
const motivations = [
  "你的激励语句",
  "另一句激励的话",
  // ...
];
```
